##  Project Overview

This repository contains a clean and well‑structured Convolutional
Neural Network (CNN) project for **image classification**.\
It is designed for learning and understanding deep‑learning concepts
step‑by‑step using **Python** and **Jupyter Notebooks**.

The notebooks included in this repository explain:

-   Convolution operation\
-   Padding and stride
-   Pooling layers
-   Flattening and fully connected layers
-   Building a complete CNN model
-   Practical implementation on **MNIST dataset**

## Dataset

The project uses the **MNIST handwritten digit dataset** consisting
of: - 60,000 training images\
- 10,000 testing images\
- 28 × 28 grayscale images
